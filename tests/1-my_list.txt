The ``MyList`` class
====================

Importing the class:
    >>> MyList = __import__('1-my_list').MyList

Testing basic sorting:
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> print(my_list)
    [1, 4, 2, 3, 5]
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]
    >>> print(my_list)
    [1, 4, 2, 3, 5]

Testing with empty list:
    >>> empty_list = MyList()
    >>> empty_list.print_sorted()
    []

Testing with negative numbers:
    >>> neg_list = MyList([-1, -5, 0, -2])
    >>> neg_list.print_sorted()
    [-5, -2, -1, 0]
